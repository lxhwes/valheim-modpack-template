Plugins Directory for valheim-modpack-template

This directory is where you can place custom plugins for your Valheim modpack.

Typical plugin types include:
- Custom DLL files for BepInEx plugins
- Custom asset bundles
- Custom scripts

When adding plugins to your modpack:
1. Ensure they are compatible with your modpack's dependencies
2. Test them thoroughly for compatibility with other mods
3. Document them in your main README.md file
4. Update your manifest.json if they require additional dependencies

Note: If you're distributing plugins created by others, make sure you have permission
and provide proper attribution in your documentation.
